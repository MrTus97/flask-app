# Hướng dẫn sử dụng

1. Cài đặt môi trường ảo

```bash
python -m venv venv
```

2. Kích hoạt môi trường ảo

```bash
.\venv\Scripts\activate
```

3. Cài đặt thư viện

```bash
(venv) pip install -r requirements.txt
```

4. Khởi động ứng dụng

```bash
(venv) python .\app.py
```

5. Sử dụng ứng dụng

```bash
> http://127.0.0.1:5000
```
